Usage Guide

  • Docker Image

    Use a specific tag for stability, e.g. redis:7-alpine. Defaults to latest if tag is omitted.

  • Port Mapping

    Format is [HostPort]:[ContainerPort]. E.g. 8080:80 maps port 8080 on Windows to port 80 inside the container.

  • Environment Variables

    Used for runtime configurations, e.g. MYSQL_ROOT_PASSWORD=secret.
